# Task Reminder App (Dart Async & Await)

## Project Overview
This project demonstrates the use of Dart programming language to build the backend logic of a simple Task Reminder Application using asynchronous programming concepts.

## Features
- Task Model using Dart Classes
- Asynchronous task loading
- Adding new tasks with async operations
- Task display functionality\
- Simulated database operations using Future.delayed()

## Technologies Used
Dart Programming Language

## Concepts Used
- Classes and Objects
- Lists
- Functions
- Future
- Async and Await

## Project Structure
task.dart → Task model class  
task_service.dart → Handles async task operations  
main.dart → Main application execution

## Learning Outcomes
- Understanding Dart fundamentals
- Implementing asynchronous programming using Future
- Using async and await in real-world scenarios
- Structuring modular Dart applications
